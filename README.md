# Fotowinnow

A web application that allows users to add customizable watermarks to their images.

## Features

- Upload single images for watermarking
- Customize watermark text
- Choose from multiple font options:
  - Courier New
  - Monaco
  - Consolas
  - Lucida Console
- Select output quality/resolution:
  - 512p
  - 1080p
  - 2K
  - 4K
- Download watermarked images in PNG format
- Built with modern web technologies:
  - Next.js 15
  - React 19
  - tRPC
  - TypeScript
  - TailwindCSS
  - Sharp for image processing

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to use the application.
